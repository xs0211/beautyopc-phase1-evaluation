# BeautyOPC 一期开发评估包

这是用于 Vercel 部署的静态网页版本。

## 入口

部署后默认打开：

```text
index.html
```

## 部署方式

1. 将本目录内的全部文件上传到 GitHub 仓库根目录。
2. 在 Vercel 中导入该 GitHub 仓库。
3. Framework Preset 选择 `Other`。
4. Build Command、Output Directory、Install Command 均保持为空。
5. 点击 Deploy。

## 注意

- 页面中的 Figma 原型为远程嵌入，对方需要拥有对应 Figma 文件的查看权限。
- Markdown 文件用于开发团队下载查看，主展示页面为 `index.html`。
